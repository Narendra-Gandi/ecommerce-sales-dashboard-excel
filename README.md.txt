# Ecommerce Sales Dashboard (Excel)

## Project Overview
This project analyzes ecommerce sales performance using an interactive Excel dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Data Visualization

## KPIs
- Total Orders
- Total Quantity Sold
- Total Sales Amount
- Customer Rating
- Average Delivery Time

## Dashboard Features
- 13-week sales trend analysis
- Popular product category analysis
- Customer purchase channel analysis
- Delivery time analysis
- Customer satisfaction insights

## Insights
- T-Shirts are the most popular product category.
- Most orders are placed through App and Website.
- Average delivery time is around 2.4 days.
